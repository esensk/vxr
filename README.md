# Visual Effects Checker

- Visual fX checkeR -> vxr

# 機能

- 画像処理用コードの出力結果確認用ツール
- 元画像と処理適用後の画像とコードを一覧で見ることができる

# 開発環境

- WSL (Ubuntu 18.04.3 LTS)

# 実行環境

- pipenv で構築

# 構成

```
vxr
├── constants.py
├── app.py
├── dao.py
├── db
│   └── sql
├── static
│   ├── css
│   └── img
└── views
```
